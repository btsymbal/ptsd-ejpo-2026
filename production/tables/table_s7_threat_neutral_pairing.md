# Table S7. Threat–Neutral Pairing Scheme

| Threat Category | Neutral Counterpart(s) | Number of Slides | Delta Computation |
|:---|:---|:---:|:---|
| angry_face | neutral_face | 10 | Threat dwell % − Neutral dwell % |
| anxiety_inducing | neutral | 14 | Threat dwell % − Neutral dwell % |
| warfare | happy_event, neutral | 12 | Threat dwell % − Neutral dwell % |
| soldiers | happy_face, neutral, neutral_face | 8 | Threat dwell % − Neutral dwell % |
| **Total** | | **44** | |

*Note.* Each slide contained one threat image and one neutral image. Delta dwell was computed as the difference between threat and neutral dwell percentages per slide. The SD of delta dwell across slides within a session was retained as the attention bias variability metric used in H3. The 44 threat–neutral slides were interspersed among 75 total slides (31 non-threat slides were omitted from delta and temporal analyses). Categories with multiple neutral counterparts (warfare, soldiers) were paired based on slide composition: the neutral image co-occurring with the threat image on each slide served as its counterpart.
